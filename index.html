Java Script

Qual é a diferença entre o JavaScript e o Java?

A linguagem de programação JavaScript, desenvolvida pela Netscape, Inc.,
não faz parte da plataforma Java.
O JavaScript não cria applets ou aplicações independentes.
Na sua forma mais comum, o JavaScript fica embutido nos documentos HTML e pode fornecer níveis de interatividade para páginas Web que não são acessíveis com um HTML simples. Diferenças-chave entre o Java e o JavaScript:
Java é uma linguagem de programação OOP,
ao passo que Java Script é uma linguagem de scripts OOP.
Java cria aplicações executadas em uma máquina virtual ou em um browser,
ao passo que o código JavaScript é executado apenas em um browser.
O código Java precisa ser compilado,
ao passo que os códigos JavaScript estão totalmente em texto.
Eles requerem plug-ins diferentes.

Quem criou e quando surgiu o Java Script?

Javascript surgiu em 1995 e foi criada por Brendan Eich,
a pedido da empresa Netscape (um dos primeiros navegadores)
com o objetivo de validar formulários HTML.
Como os browsers ainda eram estéticos,
essa linguagem foi revolucionária para a época.




Netscape

Em agosto de 1995 Netscape e Microsoft da inicio ao que ficou conehcido como a "Guerra dos Navegadores".
Durante esta guerra, o Netscape Navigator, que chegou a dominar 90% do mercado nos anos 90,
viu sua participação “derreter”, na medida em que a Microsoft lançava mão de artifícios concorrenciais “questionáveis” para promover seu Internet Explorer.
Esta “batalha sangrenta” culminaria, em 1998, com um processo judicial antitruste contra a Microsoft por concorrência desleal, movido pelo Departamento de Justiça Americano e mais 20 outros estados. Ela faria também com que a Netscape, reconhecendo que havia perdido a briga contra o navegador Internet Explorer da Microsoft,
anunciasse que seu produto Netscape Communicator passaria a ser gratuito e que ainda divulgaria publicamente seu código fonte,
fazendo nascer o projeto mozilla.org, que posteriormente daria origem ao navegador Mozilla Firefox.
Neste mesmo ano de 1998, com a Netscape tendo perdido e maior parte do seu universo de usuários, o provedor de internet America Online (AOL) adquiria a empresa, tornando-a sua subsidiária. Isso porque a AOL acreditava que, embora o navegador Netscape tenha perdido sua posição de destaque, a marca ainda era muito lembrada e forte. Mas após quase uma década mantendo o desenvolvimento do Netscape, a AOL anunciaria, em 28 de dezembro de 2007,
que deixaria de lançar novas versões do browser, para a tristeza dos seus fãs.
Isso se concretizaria em 20 de fevereiro de 2008, quando a derradeira versão Netscape Navigator 9.0.0.6 seria lançada,
encerrando definitivamente a jornada deste lendário navegador de internet.
Embora a Netscape tenha enfrentado desafios e mudanças estratégicas ao longo do tempo,
que eventualmente a levassem a encerrar suas atividades comerciais, seu impacto na história da internet é inegável,
contribuindo para a massificação da web, a introdução de diversos padrões abertos e influenciando todos os navegadores que viriam depois dele.
